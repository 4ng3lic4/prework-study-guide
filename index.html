HTML

  • The head element is a container for metadata, it contains information about the webpage
  • The meta elements contain information about the page used by the browser
  • The user won't see the information contained in the head nor meta elements.
  • The body element reprresents he visible content shown to the user.

CSS

  • CSS is a language of style rules that we use to apply styling to our HTML content,
  • The CSS code is for styling the website so it looks in an attractive and organized way
  • The CSS code can be within the HTML code or in a different style sheet
  • We can use CSS properties to create borders and spaces arounf HTML elements
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git and GitHub are the version control software and cloud hosting service used to track changes in our code
  • They store these changes as versions
  • To access Git, we use the command line
  • Git commands always start with git
  • Other Git-based repository hosting services are Bitbucket and SourceForge.
  • git status: chacks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a scripting or programming language that allows you to implement complex features on web pages
  • It is the third layer of the layer cake of standard web technologies, two of which (HTML and CSS)
  • JavaScript is the tool that takes a static HTML/CSS webpage and makes it interactive
  • JavaScript is a critical tool in every web developer's tool kit
